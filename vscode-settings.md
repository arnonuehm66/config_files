# VSCode / VSCodium Settings

## Settings

Open `Settings` with `CTRL + ,`.

* Set to `never`: `Auto Closing` ...
  - ... `Brackets`
  - ... `Comments`
  - ... `Quotes`

* Enable `Trim Trailing Whitespaces` (when saving).

* Set ruler at column 80`Editor: Rulers`. Click `Edit in settings.json` and add `"editor.rulers": [80]`.

* Add `Noto Sans Mono` to `Font Family`.

* Change `Tab Size` to 2.

* Change `inlayHints` to `off` or key-press only.

## Keyboard Shortcuts

Go to `File/Preference/Keyboard Shortcuts ...`

* Add:
  - Go Back ->   Alt+ LeftArrow
  - Go Foward -> Alt+ RightArrow
